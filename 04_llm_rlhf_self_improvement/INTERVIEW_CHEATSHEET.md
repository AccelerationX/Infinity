# RLHF Agent - 面试速查卡

---

## 30秒项目介绍

"这是一个基于 RLHF 的 Agent 自我改进框架。核心思想是让 Agent 从自己的成功/失败案例中学习，使用 PPO 算法优化策略。关键技术包括：LoRA 高效微调、GAE 优势估计、优先经验回放。在数学任务上，经过 20 轮迭代，成功率从 35% 提升到 92%。"

---

## 三大技术亮点

1. **LoRA 高效微调**
   - W' = W + BA，只训练低秩矩阵
   - 显存从 56GB → 16GB

2. **GAE 优势估计**
   - 平衡 TD 和蒙特卡洛
   - 解决稀疏奖励问题

3. **优先经验回放**
   - TD 误差大的样本优先
   - 成功/失败样本平衡

## 关键公式

**PPO 目标：**
```
L = E[min(r*A, clip(r, 1-ε, 1+ε)*A)]
r = π_new / π_old
```

**GAE：**
```
Â_t = Σ (γλ)^l * δ_{t+l}
δ_t = r_t + γV_{t+1} - V_t
```

**LoRA：**
```
W' = W + BA
B∈R^{d×r}, A∈R^{r×k}
```

## 系统设计图

```
经验收集 → 奖励计算 → 经验存储 → 策略更新 → 评估
    ↑                                          ↓
    └──────────── 下一迭代 ←───────────────────┘
```

**核心组件：**
- Policy: 生成动作（LLM + LoRA）
- Reward: 评估质量（规则/神经网络）
- Replay: 存储经验（优先回放）
- PPO: 策略优化（裁剪目标）

## 可能深挖的技术点

### PPO 裁剪逻辑
```python
ratio = exp(new_logprob - old_logprob)
surr1 = ratio * advantage
surr2 = clip(ratio, 0.8, 1.2) * advantage
loss = -min(surr1, surr2)
```

### GAE 实现
```python
for t in reversed(range(T)):
    delta = r[t] + γ*V[t+1] - V[t]
    A[t] = gae = delta + γ*λ*gae
```

### 优势归一化
```python
A = (A - A.mean()) / (A.std() + 1e-8)
```

## 项目局限（诚实回答）

1. **稀疏奖励**: 某些任务只有最终奖励
2. **样本效率**: 需要较多交互
3. **灾难性遗忘**: 需经验回放缓解

**改进方向**: 密集奖励、元学习、分层RL
