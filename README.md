# Research Projects

This repository contains a collection of research and engineering projects spanning AI Agent systems, quantitative finance, and financial technology.

---

## Projects

### AI / Agent Systems

| # | Project | Description |
|---|---------|-------------|
| 01 | [LLM Agent Capability Benchmark](01_llm_agent_capability_benchmark/) | Multi-dimensional evaluation framework for LLM Agents with automated scoring and failure attribution. |
| 02 | [LLM Agent Security Sandbox](02_llm_agent_security_sandbox/) | Dynamic permission grading and sandbox isolation for safe Agent code/tool execution. |
| 03 | [Multi-Agent Collaboration Protocol](03_multi_agent_collaboration_protocol_v2/) | Dynamic role assignment, structured messaging, and consensus mechanisms for multi-agent teams. |
| 04 | [RLHF Self-Improvement](04_llm_rlhf_self_improvement/) | PPO-based continuous policy optimization for Agents using reward models and experience replay. |
| 05 | [LLM Inference Optimization](05_llm_inference_optimization/) | Quantization, speculative decoding, KV-cache optimization, and edge deployment research. |
| 13 | [Computer Use Agent](13_computer_use_agent/) | UIA-based desktop automation Agent with atomic execution and visual verification. |
| 14 | [MCP Tool Hub](14_mcp_tool_hub/) | Production-grade tool routing hub using the official MCP SDK with semantic matching and DAG planning. |
| 15 | [Agent Lifelong Learning](15_agent_lifelong_learning/) | Skill discovery, distillation, and failure replay for continuous Agent self-improvement. |
