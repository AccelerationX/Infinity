# Agent Companion Benchmark (ACB) 模型测评报告

> **测试时间**: 2026-04-13  
> **测试框架**: Agent Companion Benchmark v1.0  
> **对比模型**: Kimi K2.5, MiniMax M2.7, DeepSeek V3.2

---

## 📊 执行摘要

本报告基于 **Agent Companion Benchmark (ACB)** 框架，对三款主流中文大模型进行了系统性评测。ACB 是专为 **AI Companion / AI 伴侣系统** 设计的评测体系，涵盖通用能力与 Companion 专属能力两大维度。

### 测试配置
| 项目 | Kimi K2.5 | MiniMax M2.7 | DeepSeek V3.2 |
|------|-----------|--------------|---------------|
| **API 提供商** | Moonshot | MiniMax Token Plan | DeepSeek |
| **Base URL** | api.moonshot.cn/v1 | www.minimaxi.com/v1 | api.deepseek.com |
| **Model ID** | kimi-k2.5 | MiniMax-M2.7 | deepseek-chat |
| **Temperature** | 1.0* | 1.0 | 0.7 |
| **Max Tokens** | 2048 | 2048 | 2048 |
| **测试任务数** | 20** | 26 | 26 |

\* Kimi K2.5 仅支持 temperature=1  
\*\* Kimi 测试因响应超时未完成全部任务，数据基于 20 个 Companion 任务

## 🏆 总体排名

| 排名 | 模型 | 综合得分 | 成功率 | 平均响应时间 |
|------|------|----------|--------|--------------|
| 🥇 1 | **DeepSeek V3.2** | **0.685** | **69.2%** | **6.35s** ⭐ |
| 🥈 2 | **MiniMax M2.7** | **0.729** | **73.1%** | **21.58s** |
| 🥉 3 | **Kimi K2.5** | **0.754** | **85.0%** | **32.72s*** |

\* Kimi 数据基于 Companion 任务（不含通用任务），可能偏高

### 关键发现
1. **DeepSeek V3.2**: 响应速度最快（6.35s），综合表现均衡，性价比最高
2. **MiniMax M2.7**: 成功率最高之一（73.1%），但响应较慢（21.58s），规划任务耗时明显
3. **Kimi K2.5**: 在 Companion 任务上表现优秀（85% 成功率），但响应最慢（32.72s）

## 📝 测试局限性说明

1. **Kimi K2.5 数据不完整**: 因响应超时（>600s），未获得全部 26 个任务数据，当前数据基于 20 个 Companion 任务，可能高估其综合得分

2. **温度参数差异**: Kimi 强制 temperature=1，其他模型使用 0.7，可能影响输出质量和一致性

3. **API 稳定性**: 测试期间各模型 API 稳定性不同，MiniMax Token Plan 有速率限制，可能影响测试结果

4. **工具调用差异**: 不同模型对工具调用的支持程度不同，MiniMax 不支持原生 tools 参数，采用 prompt 注入方式

5. **评测任务局限**: ACB 任务集仍在完善中，当前 26 个任务可能无法完全覆盖所有 Companion 场景
